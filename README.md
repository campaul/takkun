# Takkun
Takkun is a text editor written from scratch for educational purposes.

## Commands
- `ctrl + t` Create a new file
- `ctrl + o` Open a file
- `ctrl + s` Save the current file
- `ctrl + x` Close the current file
- `ctrl + q` Quit
- `ctrl + n` Next file
- `ctrl + p` Previous file
- `ctrl + f` Find
